This schema contains LINX XML (which also references xml elements from the GB taf_cat_Complete.xsd) for the following LINX Messages:

MMESSAGE						MESSAGE NAME					MESSAGE DESCRIPTION
Passenger Train Consist			PassengerTrainConsistMessage	This message is for passenger train allocation and consists.
Freight Train Consist			FreightTrainConsistMessage		This message is for freight train consists.
Crew Allocation					CrewAllocationMessage			This message is Train Centric whereby crew members are allocated to a train journey or train journey leg for a single train service.

O - Vehicle overdue Mandatory PPM. (TOPS-generated)
P - Vehicle Due Planned Preventative Maintenance
Q - Defective Brake - Brakepipe NOT operative. Vehicle may not go forward with defective brake and

T - Defective Vehicle. Requires Transhipment and/or load adjustment prior to going forward to repair point
U - Vehicle Due for VIBT. (TOPS-generated)
W - Vehicles for Modification, Conversion or Overhaul
